HomegoingHQ — Live Guide (preview)

A gentle, face-to-face companion to help a family settle an estate. This preview uses the LiveAvatar embed: a real-time avatar answers, steered by the compassionate context you configure.

Please read before using with families. This preview runs LiveAvatar FULL mode — HeyGen's model answers, guided by your context, with HeyGen's moderation. It does not run the ai-guide function's Survivor-Mode guardrails (information not advice; defer contested legal, tax, and financial questions to a licensed professional). Configure a strict, grief-appropriate context/persona in the LiveAvatar dashboard that mirrors those rules. Keep Sandbox mode on while testing — it's free (no credits).
Set up

Get a LiveAvatar API key at app.liveavatar.com/developers (separate from your HeyGen key) and set it in Netlify as LIVEAVATAR_API_KEY. Create a warm, calm avatar and a context/persona in the LiveAvatar dashboard, then paste their IDs below. The pre-filled IDs are a public sandbox demo so you can see it work first.
